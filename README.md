# mini_calendar

Date component developed with Flutter, plans to support display, swipe left and right, add date mark, radio, display week, etc.

Flutter开发日历组件。

- [√] 展示日历周视图
- [√] 支持左右滑动切换月视图
- [√] 支持添加自定义Marker
- [ ] 支持农历和阳历显示
- [ ] 支持添加自定义事件
- [ ] 支持自定义起始周
- [ ] 支持自定义日期控件
- [ ] 优化性能
- ……
